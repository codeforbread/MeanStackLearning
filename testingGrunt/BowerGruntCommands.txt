##### To install Bower globally #####

sudo npm install -g bower


##### To install Bower locally #####

sudo npm install bower


##### To install grunt cli globally #####

create one folder sudo mkdir testingGrunt folder
cd testingGrunt/
sudo npm install -g grunt-cli


##### To install grunt locally #####

cd testingGrunt/
sudo npm install grunt

create file Gruntfile.js and export module grunt
